# SOP-003: Data Integrity and Database Operations
Version: 1.0
Classification: Confidential
Last Updated: 2024-03-15

## Schema Mismatch

If schema mismatch occurs:
1. Validate schema version
2. Roll back migration if required
3. Assign to DBA-Team

## Data Loss Incident

If data loss is reported:
1. Check backup status
2. Restore latest valid snapshot
3. Notify Business stakeholders

## Query Performance Issues

If query latency is high:
1. Check indexing
2. Analyse execution plan
3. Tune queries
